# Resume Highlights: Stock Prediction AI

Here are a few ATS-friendly (Applicant Tracking System) ways to include this project on your resume, tailored to different space constraints. Pick the option that best fits your resume structure!

### Option 1: Detailed (3-4 bullets)
**Quantitative Developer / AI Researcher Project**
* **Stock Prediction AI Framework**: Engineered an end-to-end Python timeframe forecasting pipeline integrating numerical financial data and NLP sentiment analysis via yfinance and VADER.
* **Hybrid Deep Learning Model**: Architected a high-accuracy CNN-LSTM neural network using TensorFlow/Keras to extract sequential technical features (RSI, MACD, Bollinger Bands) from time-series data.
* **Algorithm & Hyperparameter Optimization**: Implemented dynamic early stopping, model checkpointing, and Adam optimizer learning rate scheduling, significantly improving Directional Accuracy. 
* **Backtesting Engine**: Built a customized evaluation framework utilizing MAPE, RMSE, Sharpe Ratio, and Maximum Drawdown metrics to empirically validate the model's predictive power against real-world market movements.

### Option 2: Concise (2 bullets)
**AI Stock Predictor Pipeline**
* Developed a high-performance quantitative stock prediction engine in Python, combining a bespoke CNN-LSTM neural network with RSI, MACD, and NLP-based fundamental sentiment analysis.
* Evaluated time-series predictions through a custom backtesting suite computing Sharpe Ratio and Maximum Drawdown, optimizing Mean Absolute Percentage Error (MAPE). 

### Option 3: Skills Based
**Personal Project: Financial Deep Learning Pipeline**
* **Tech Stack**: Python, TensorFlow/Keras, Pandas, scikit-learn, yfinance, NLTK
* Engineered a CNN-LSTM deep learning architecture to forecast asset movements by correlating 60-day historical time-series sequences, technical indicators, and NLP sentiment scores.
