# Image Instructions

## Adding Your Founder Image

To add your image as the founder of KV Store:

1. Save your image file as `founder.jpg`
2. Place it in the `static/images/` folder
3. The image will automatically appear on the About page

### Image Requirements:
- Format: JPG/JPEG
- Recommended size: 400x400 pixels or larger
- The image will be automatically resized and cropped to fit

### Current Location:
- The image is referenced in `templates/about.html`
- If the image is not found, a default user icon will be displayed

### File Path:
```
static/images/founder.jpg
```
